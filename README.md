# System-Control-App

A simple Tkinter-based Python application that provides essential system control options for Windows, including shutdown, restart, delayed restart, and log out. This app offers a user-friendly GUI for executing common system actions without using the command line.

---

## **Features**

- Shutdown – Turn off the PC immediately.
- Restart – Restart the PC immediately.
- Restart in Time – Schedule a restart after a specified delay (default: 20 seconds).
- Log Out – Sign out the current user session without shutting down the PC.
---

## **Requirements**

- Python 3.x
- Tkinter (usually included with Python on Windows)
  
---

## **Installation**

1. Clone or download the repository.
2. Ensure Python 3.x is installed.
3. Run the application:
```
python system_control_app.py
```
> Note: On Windows, running shutdown or restart commands may require administrator privileges.

---

## **Usage**

1. Launch the application.
2. Click on the desired action button:

Shutdown – Powers off your PC.
Restart – Immediately restarts the PC.
Restart in Time – Waits 20 seconds and then restarts.
Log Out – Ends the current user session.

---

## **Safety Note**

Use the app responsibly. Unsaved work will be lost if the PC shuts down or restarts.

Administrator privileges may be required for some actions.
